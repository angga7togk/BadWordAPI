# BadWordAPI
BadWord API for java

# Maven
add in your application
### Repository
```xml
    <repositories>
        <repository>
            <id>jitpack.io</id>
            <url>https://jitpack.io</url>
        </repository>
    </repositories>
```
### Dependency
```xml
    <dependency>
        <groupId>com.github.angga7togk</groupId>
        <artifactId>BadWordAPI</artifactId>
        <version>1.0.0</version>
    </dependency>
```
